# AOS OpenAPI Specifications Repository

This repository hosts the OpenAPI specifications for the APIs managed by our organization. The specifications are organized into JSON and YAML formats for better accessibility and usage.

## Repository Structure

- `/aos`
    - `/json`: Contains OpenAPI specifications in JSON format.
    - `/yaml`: Contains OpenAPI specifications in YAML format.

## Usage

The OpenAPI specifications housed in this repository serve two primary purposes:

1. **Postman Collections Generation**:
   - The OpenAPI files can be imported into Postman to generate collections for API testing and exploration.

2. **API Documentation Viewing**:
   - The specifications can be used with tools like Swagger to visualize the API documentation, providing an interactive interface for exploring the available API methods and operations.
